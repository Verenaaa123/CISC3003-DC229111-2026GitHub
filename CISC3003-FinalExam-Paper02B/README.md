This Scenario B project demonstrates:

- HTML contact form with browser validation
- PHPMailer configuration in `php/mail_config.php`
- email sending in `php/send_contact.php`
- debug output in `debug.php` and `logs/email-debug.log`
- post / redirect / get pattern after form submission

Import `db/database.sql`, then open `php/index.php`. The root `index.html` redirects to `php/index.php`. PHPMailer is included under `vendor/phpmailer/phpmailer`; edit SMTP values before sending real email.
